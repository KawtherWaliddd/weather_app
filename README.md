# 🌤️ Weather App

A simple and clean weather application built with Flutter that allows users to view the current weather and a 4-day forecast for any city.

##  Features

-  Search weather by city name
-  Display current weather (temperature, description, icon)
-  4-day weather forecast
-  Location-based weather
-  Responsive and user-friendly UI


## Tech Stack

- **Flutter** 
- **Dart**
- **Dio package** for API requests
- OpenWeatherMap API
