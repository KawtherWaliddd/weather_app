# 🌦️ Weather App

A simple and clean weather app built with Flutter, providing real-time weather data and a 4-day forecast. The app allows users to search for any city and view detailed weather information with a responsive and attractive UI.

## Features

-  Search for weather by city name
-  Display current temperature, weather condition, and icon
-  Light and Dark mode support
-  Pull-to-refresh weather data
-  Responsive design for different screen sizes

## Tech Stack

- **Flutter** – UI toolkit for building natively compiled applications
- **Dio** – Powerful HTTP client for Dart
- **Bloc / Cubit** – State management
